# MERN Authentication System (Login, Signup, Protected Routes)

This project is a complete MERN-stack authentication system built using:

- **React (Frontend)**
- **Node.js + Express (Backend)**
- **MongoDB Atlas (Database)**
- **JWT Authentication**
- **Protected Routes using React Router**

Users must log in to access the **Home** page. If not authenticated, they will be redirected to the **Login** page.

---

## :wheel_of_dharma: Features

### **Frontend (React)**
- Signup page
- Login page
- JWT-based authentication
- Protected Home page
- Private Route implementation
- Logout functionality
- Toast notifications
- Fetching products after authentication

### **Backend (Node.js + Express)**
- User Signup API
- User Login API
- JWT token generation
- Middleware for validating tokens
- Secure Product API
- MongoDB Atlas connection

---

## :hammer_and_pick: Tech Stack

### **Frontend**
- React
- React Router v6
- React Toastify
- Fetch API
- LocalStorage for token handling

### **Backend**
- Node.js
- Express.js
- Mongoose
- JSON Web Tokens (JWT)
- bcrypt
- dotenv
- MongoDB Atlas
